# Data Mining Final Project

## Overview

This project focuses on using different data mining techniques to better understand patterns related to diabetes risk. The dataset includes health indicators such as BMI, age, physical activity, and medical conditions like high blood pressure and cholesterol.

The goal of this project was to apply multiple methods including data cleaning, exploratory data analysis, regression, classification, clustering, and association rule mining to gain meaningful insights from the data.


## Dataset

The dataset used in this project comes from the Behavioral Risk Factor Surveillance System. It contains information about individuals and their health conditions and lifestyle habits.

This dataset was chosen because it represents a real world problem and includes a wide range of features that can help explain diabetes risk.


## Process

The project was completed in several steps.

First, the data was cleaned and prepared by checking for missing values, removing duplicates, and making sure everything was consistent.

Next, exploratory data analysis was performed using visualizations to understand patterns and relationships between variables.

After that, different models were built. Regression was used to explore relationships between variables. Classification models such as Decision Tree and k Nearest Neighbors were used to predict diabetes. Clustering was used to group individuals based on similar characteristics. Association rule mining was used to find patterns between different health conditions.


## Key Insights

The results showed that the Decision Tree model performed better than k Nearest Neighbors when predicting diabetes.

Clustering revealed that individuals can be grouped into different health profiles, with some groups showing more risk factors than others.

Association rule mining showed that conditions like high blood pressure and high cholesterol are often linked with diabetes, suggesting that multiple factors contribute to the condition.


## Challenges

One challenge was working with a large dataset, which required careful handling to make sure the analysis ran smoothly.

Another challenge was choosing the right models and parameters. This was addressed by testing different approaches and using tuning techniques to improve performance.

Understanding and interpreting the results, especially from clustering and association rules, also required some trial and error.


## Conclusion

Overall, this project helped show how different data mining techniques can be used together to better understand a real world problem.

It also showed how data can be used to support decision making, especially in areas like healthcare where identifying risk early can make a big difference.
